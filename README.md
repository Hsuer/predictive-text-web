## 英文写作助手 - Web前端部分

### 后端
[Predictve-Text](https:github.com/jeffylu/predictive-text)

### 使用方法
在 [config.js](predictive-text-web/src/utils/config.js) 中修改后端接口。

### 安装依赖
```
$ npm install
```

### 启动
```
$ npm start
```
